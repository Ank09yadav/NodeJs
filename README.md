##  Project Overview

This repository contains a suite of Node.js tools designed to demonstrate modern backend development practices using **ES Modules**.

**Key Features:**
*   **Create Folders & Files:** Quickly generate new directories and standard files.
*   **Write Content:** Append or write text directly to files from the CLI.

## ğŸ› ï¸ Installation & Usage

Follow these steps to set up and run the project locally.

### Prerequisites
*   [Node.js](https://nodejs.org/) (v14 or higher recommended)
*   npm (Node Package Manager)

### Quick Start

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/Ank09yadav/NodeJs.git
    cd NodeJs
    ```

2.  **Install Dependencies:**
    This project uses `chalk` for styling. Install it using npm:
    ```bash
    npm install
    ```

3.  **Run the File Manager:**
    To start the interactive file manager:
    ```bash
    node fileManager.js
    ```
    _Follow the on-screen prompts to manage your files._

4.  **Run the System Monitor:**
    To start monitoring your system resources:
    ```bash
    node systemMonitor.js
    ```
    _Press `Ctrl + C` to stop the monitor._
